I. HOW TO RUN THE PROGRAM.

Windows
1.Open the IDLE(Python GUI) for python 3 or Jupyter-notebook.
2.Specify the path of the corpus directory in the PWD variable in the code
3.a)Click Run Module in the option in IDLE editor
	OR
3.b)Open the command prompt and change directory to the IR_PROJECT
4.Please wait for compilation as it takes ~ 1-2 min for indexing a 1MB corpus

Note- You need NLTK package for successful compilation as well as the NLTK-Corpus
