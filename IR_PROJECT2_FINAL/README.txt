SVD AND CUR DECOMPOSITION

I. HOW TO RUN THE PROGRAM.

Windows
1.Open the IDLE(Python GUI) for python 3.
2.Specify the path of the CSV file in the code
3.a)Click Run Module in the option in IDLE editor
	OR
3.b)Open the command prompt and change directory to the IR_PROJECT
4.Please wait for compilation as it takes  3~4 min for decomposing a 672*9000 matrix

Note- You need SciPy,NumPy package for successful compilation 

5. SVD FILES: base.py,svd.py,test_svd.py
6. CUR FILES: CUR_NEW.py also ( cur.py,test_cur.py)
